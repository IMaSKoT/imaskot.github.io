## 3. Трансформация данных

### Преобразования типов

**Pclass → строки:**
```python
df['Pclass'] = df['Pclass'].map({1: 'F', 2: 'S', 3: 'T'})
```
Числовые классы заменены на читаемые буквенные метки.

**Sex → числа:**
```python
df['Sex'] = df['Sex'].map({'male': 0, 'female': 1})
```
Необходимо для математических операций.

### Созданные признаки

| Признак | Описание | Способ создания |
|---|---|---|
| `Age_group` | Возрастная группа | `pd.cut()` |
| `Letter` | Палуба каюты (первая буква) | `str[0]` |
| `Title` | Титул (Mr, Mrs, Miss…) | `str.extract()` + регулярка |
| `FamilySize` | Размер семьи | `SibSp + Parch + 1` |
| `isAlone` | Едет один (0/1) | `(FamilySize == 1).astype(int)` |

**Title** - 16 уникальных значений: Mr (517), Miss (182), Mrs (125), Master (40), Dr (7), Rev (6) и др.

**Winsorization для Fare:** значения выше 95-го перцентиля заменены на перцентиль.

**Age:** экстремальные значения заменены на 95-й перцентиль (54.0).

После обработки:
