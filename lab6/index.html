    Лабораторная работа №6: Очистка и трансформация данных. Pandas

    Ссылка на ipynb-борд

    Введение

    Цель работы

    Освоение методов очистки и трансформации данных с использованием библиотеки pandas на примере реальных данных из Kaggle (датасет Titanic).

    Описание набора данных

    Датасет Titanic содержит информацию о пассажирах корабля. Всего 891 запись и 12 признаков:

    Признак Описание
    PassengerId Идентификатор пассажира
    Survived Выжил ли (0 = нет, 1 = да)
    Pclass Класс билета (1, 2, 3)
    Name Имя
    Sex Пол
    Age Возраст
    SibSp Кол-во братьев/сестёр/супругов на борту
    Parch Кол-во родителей/детей на борту
    Ticket Номер билета
    Fare Стоимость билета
    Cabin Номер каюты
    Embarked Порт посадки (S, C, Q)

    1. Первичный анализ

    Статистика датафрейма

    Вывод df.info() показал: - 891 запись, 12 столбцов - Типы: int64 (5), float64 (2), object (5) - Age имеет 714 непустых значений (177 пропусков) - Cabin имеет только 204 непустых значения (687 пропусков)

    Вывод df.describe():

    Age Fare
    count 714 891
    mean 29.70 32.20
    std 14.53 49.69
    min 0.42 0.00
    50% 28.00 14.45
    max 80.00 512.33

    Особенности: стандартное отклонение Fare (≈49.7) значительно превышает среднее (≈32.2), что говорит о большом разбросе и наличии выбросов.

    Визуализация пропусков

    Вывод df.isnull().sum(): 

    Age         177
Cabin       687
Embarked      2

    Пропуски обнаружены в трёх столбцах. Cabin пропущен почти в 77% случаев.

    2. Обработка пропусков

    Методы заполнения

    Age: - Среднее: 29.7, медиана: 28.0 - Пропуски заполнены медианой (28.0), так как медиана устойчива к выбросам - Создан новый признак Age_group по диапазонам: child (0–12), teenager (12–18), adult (18–60), senior (60–100)

    Embarked: - Наиболее частое значение: S (646 раз из 891 после заполнения) - Пропуски (2 строки) заполнены модой - значением S - После заполнения: df['Embarked'].isnull().sum() == 0

    Cabin: - Слишком много пропусков (77%) для заполнения - Из первой буквы каюты извлечён признак Letter (палуба корабля) - Пропуски заменены значением 'Unknown'

    Результаты обработки

    Столбец До После
    Age 177 пропусков 0
    Embarked 2 пропуска 0
    Cabin 687 пропусков не заполнялся
    Letter - создан новый признак

    3. Трансформация данных

    Преобразования типов

    Pclass → строки: 

    df['Pclass'] = df['Pclass'].map({1: 'F', 2: 'S', 3: 'T'})
    Числовые классы заменены на читаемые буквенные метки.

    Sex → числа: 

    df['Sex'] = df['Sex'].map({'male': 0, 'female': 1})
    Необходимо для математических операций.

    Созданные признаки

    Признак Описание Способ создания
    Age_group Возрастная группа pd.cut()
    Letter Палуба каюты (первая буква) str[0]
    Title Титул (Mr, Mrs, Miss…) str.extract() + регулярка
    FamilySize Размер семьи SibSp + Parch + 1
    isAlone Едет один (0/1) (FamilySize == 1).astype(int)

    Title - 16 уникальных значений: Mr (517), Miss (182), Mrs (125), Master (40), Dr (7), Rev (6) и др.

    4. Обработка выбросов

    Выявленные выбросы

    Boxplot признака Fare показал большое количество выбросов выше верхнего уса.

    IQR-метод для Fare: - Q1 = 7.91, Q3 = 31.00, IQR = 23.09 - Нижняя граница: −26.72 (отрицательных значений нет) - Верхняя граница: 65.63 - Выбросов: 116 значений выше границы

    Методы обработки

    Winsorization для Fare: значения выше 95-го перцентиля заменены на перцентиль.

    Age: экстремальные значения заменены на 95-й перцентиль (54.0).

    После обработки: - Максимум Fare стал ≈ 112.08 (вместо 512.33) - Максимум Age стал 54.0 (вместо 80.0)

    5. Агрегация и анализ

    Среднее выживание по классам

    Класс Выживаемость
    F (1-й) 63.0%
    S (2-й) 47.3%
    T (3-й) 24.2%

    Вывод: пассажиры первого класса выживали значительно чаще.

    Группировка по классу и полу

    Класс Пол (0=муж, 1=жен) Кол-во
    F 0 122
    F 1 94
    S 0 108
    S 1 76
    T 0 347
    T 1 144

    Медианный возраст по портам посадки

    Порт Медиана возраста
    C 28.0
    Q 28.0
    S 28.0

    Особенность: все три значения совпадают - это следствие заполнения пропусков медианой (28.0), которая «перетянула» результаты. В оригинальных данных между портами были различия.

    Сводная таблица выживаемости по новым признакам

    Таблица по Age_group и isAlone:

    Age_group Не один (0) Один (1)
    child 58.2% 50.0%
    teenager 50.0% 35.3%
    adult 48.9% 30.2%
    senior 25.0% 22.2%

    Вывод: пассажиры с семьёй выживали чаще, чем одиночки - во всех возрастных группах.

    6. Метрики качества очистки

    Процент заполненных значений

    Признак % заполнен
    Age, Survived, Sex, … 100%
    Cabin 22.9%
    Title 99.89%

    Cabin остался частично пустым - он не заполнялся, вместо этого создан признак Letter. Title имеет 1 пропуск - нестандартный формат имени не распознала регулярка.

    Уникальные значения в категориальных признаках

    Признак Уникальных значений
    Pclass 3
    Sex 2
    Embarked 3
    Age_group 4
    Title 16
    Letter 9

    Корреляция между новыми признаками

    Survived FamilySize isAlone
    Survived 1.00 0.017 −0.20
    FamilySize 0.017 1.00 −0.69
    isAlone −0.20 −0.69 1.00

    Выводы: - isAlone и FamilySize ожидаемо сильно отрицательно коррелируют (−0.69) - это логично, т.к. один признак вычислен из другого - isAlone умеренно отрицательно коррелирует с выживаемостью (−0.20) - одинокие пассажиры выживали реже - Категориальные признаки (Age_group, Title, Letter) исключены из корреляционного анализа, так как содержат текстовые значения

    Заключение

    Результаты работы

    В ходе лабораторной работы был проведён полный цикл предобработки данных:

    1. Первичный анализ - изучены типы, статистики, выявлены пропуски в Age (177), Cabin (687), Embarked (2)
    2. Обработка пропусков - Age заполнен медианой, Embarked - модой, из Cabin создан новый признак Letter
    3. Трансформация - создано 5 новых признаков: Age_group, Letter, Title, FamilySize, isAlone; Sex переведён в числовой формат; Pclass - в строковый
    4. Обработка выбросов - применена winsorization для Fare и Age с помощью 95-го перцентиля; выявлено 116 выбросов в Fare методом IQR
    5. Агрегация - показана связь выживаемости с классом и полом; найдено, что пассажиры с семьёй выживали чаще
    6. Метрики качества - процент заполненных значений, распределения новых признаков, корреляции

    Выводы

    • Заполнение пропусков медианой сглаживает различия между группами - об этом свидетельствует одинаковый медианный возраст по всем портам
    • Первый класс и наличие семьи значимо связаны с выживаемостью
    • Признак isAlone оказался умеренно информативным (корреляция −0.20 с Survived)
    • Большинство пассажиров (730 из 891) - взрослые (18–60 лет)
